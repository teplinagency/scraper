Making Requests

There are two simple ways in which you can send GET requests to Scraper API (you can also send POST/PUT requests):

Send Requests to the API Endpoint

Scraper API exposes a single API endpoint for you to send GET, POST or PUT requests. For most use cases, you simply need to send a GET request to http://api.scraperapi.com with two query string parameters and the API will return the HTML response from that URL:

  • api_key which contains your API key, and
  • url which contains the url you would like to scrape
