Additional Functionality

Geotargeting

Easily geotarget 12 countries, with 50 more available upon request, by adding &country_code= to your request so you can get accurate and localized information.

JS Rendering

Render on-page javascript by simply adding &render=true to your request so you don’t have to worry about using your own headless browsers.

Residential Proxies

Seamlessly switch to residential proxies by adding &premium=true to your request so you don’t need to deal with multiple proxy venders.

JSON Autoparsing

Let use handle the parsing of the data by adding &autoparse=true to your request. We will return the data in JSON format. Only certain domains supported.

Custom Anti-Bot Bypasses

Need to scrape a website that is protected by a difficult anti-bot, then let our team develop a custom anti-bot bypass that allows you to scrape the data you need.
